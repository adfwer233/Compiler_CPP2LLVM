# 词法分析部分

我们的词法分析器通过 Antlr 4 自动生成， 语法文件位于 `/grammar/MyCPPLexer.g4` 中

语法文件主要包括如下部分

- 关键字
- 算符
- 标识符
- 整数
- 浮点数
- 字符和字符串
- 预处理声明

antlr 会生成一个词法分析器中，我们使用 Python 作为目标语言，在根目录下的 `main.py` 中，我们调用这个词法分析器，接收输入文件并输出 token 流，测试文件位于 `test/lexer_test.cpp` 中，词法分析结果如下。
